## Base URL
https://ticketguru.com

## EventRecord

**Method:** `GET`

        "event_endtime": "2023-01-01T23:23:00",
        "deleted": false
    },
    {
        "eventrecord_id": 2,
        "eventrecord_name": "Tapahtuma 2",
        "venue": "Paikka 2",
        "city": "Kaupunki 2",
        "ticketsmax": 200,
        "event_starttime": "2023-02-02T12:22:00",
        "event_endtime": "2023-02-02T00:24:00",
        "deleted": false
    },
    {
        "eventrecord_id": 3,
        "eventrecord_name": "Tapahtuma 3",
        "venue": "Paikka 3",
